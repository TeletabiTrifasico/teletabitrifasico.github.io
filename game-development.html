Infinite Runner Game

Description:
This project is a complete 2.5D infinite runner game built in Unreal Engine 5.6. The game challenges players to avoid obstacles, collect pickups, and survive for as long as possible on a procedurally generated track. The focus was on developing dynamic, replayable gameplay and mastering modular level design, animation systems, and real-time object spawning.

Key Features:

  • Procedural level generation with modular tile sets
  • Player character with animated states (idle, run, jump, die)
  • Dynamic enemy and collectible spawning
  • Real-time scoring system with UI updates
  • Game-over detection and screen transitions
  • Combination of 2D sprites and 3D environments for hybrid visual style

Racing Game

Description:

Key Features:

  • Driveable car with arcade handling
  • Checkpoint and lap-tracking system
  • Custom race UI with timer and lap count
  • Start/finish logic and win condition
  • Multiple vehicle rigging setup (including Blender integration)
  • Responsive input controls for keyboard/gamepad
