Infinite Runner Game

Unreal Engine 5
GitHub

Description:
This project is a complete 2.5D infinite runner game built in Unreal Engine 5 using Blueprints. The game challenges players to avoid obstacles, collect pickups, and survive for as long as possible on a procedurally generated track. The focus was on developing dynamic, replayable gameplay and mastering modular level design, animation systems, and real-time object spawning.

Key Features:

  • Procedural level generation with modular tile sets
  • Player character with animated states (idle, run, jump, slide)
  • Dynamic enemy and collectible spawning
  • Real-time scoring system with UI updates
  • Game-over detection and screen transitions
  • Combination of 2D sprites and 3D environments for hybrid visual style

Game 05

Turn Based RPG Game

Unreal Engine 5
GitHub

Description:
This project is a turn-based 2.5D RPG created in Unreal Engine 5.6. The game focuses on strategic, turn-based combat between player-controlled characters and enemy units, incorporating magic attacks, healing mechanics, initiative sorting, and dynamic UI feedback. It served as an excellent exercise in building combat systems, gameplay flow, and clean state-driven logic for RPGs.

Key Features:

  • Custom Player & Enemy Classes: Blueprint-based character data structure with health, damage, and other stats
  • Bubble Sort for Initiative: Initiative order logic based on speed stats, implemented with Bubble Sort for turn management
  • Turn Management System: Fully dynamic turn-sequencing logic ensuring alternating player/enemy actions
  • Projectile & Magic Mechanics: Implementation of animated projectile attacks like Magic Missile including animations
  • Healing System: Action-based healing mechanics with dedicated logic for recovery, including SFX
  • UI Systems: Intuitive, reactive interfaces for selecting actions, health bars, and game states
  • Enemy AI: Automated enemy behavior based on available actions and real-time turn decisions
  • Combat Feedback: Damage/heal visual feedback, character death handling, game-over screen, and SFX integration
